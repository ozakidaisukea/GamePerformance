
●制作期間 			2018年6月〜9月
●プラットフォーム	DirectX Ver9.0
●制作ソフト		VisalStudio2015
●使用言語			C++
●動作環境			Windows10
●プレイ人数		1人

●操作方法			//===========================================
					// キーボード版
					//===========================================
					Wキーで上移動
					Aキーで左移動
					Sキーで下移動
					Dキーで右移動
					SPACEキーで弾発射
					Pキーでポーズ
					十字キーでポーズ選択
					ENTERキーでセレクト
					Qキー＆Eキーで必殺技
					
					//===========================================
					// ゲームパッド版(Logicool Gamepad F310)
					//===========================================
					十字キーまたは左スティックで移動
					Aボタンでセレクトまたは弾発射
					LBボタン＆RBボタンで必殺技
					STARTボタンでポーズ、セレクト
