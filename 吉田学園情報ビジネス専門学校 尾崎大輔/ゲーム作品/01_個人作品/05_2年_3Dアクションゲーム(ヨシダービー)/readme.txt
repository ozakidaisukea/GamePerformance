●操作方法			//===========================================
					// キーボード版
					//===========================================
					[タイトル、チュートリアル、リザルト]
					 ENTERキーでセレクト
					 
					[チュートリアル]
					 SPACEキー長押しで出馬表
					 
					[ゲーム]
					 ENTERキーでダッシュ
					
					//===========================================
					// ゲームパッド版(Logicool Gamepad F310)
					//===========================================
					STARTボタンでセレクト
					LBボタン、RBボタンでダッシュ
